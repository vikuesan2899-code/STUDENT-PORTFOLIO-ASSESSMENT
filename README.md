## Project Description

This project is a responsive personal portfolio website developed as part of the COS106 Web Technologies assessments. It showcases my profile, projects, academic planner, and contact information using HTML, CSS and JavaScript.

## Features

- Home Page
- About Me Page
- Projects Page
- Academic Planner
- Contact Form with Validation
- Responsive Design
- Local Storage for Tasks
- Multimedia Support (Images and Video)

## Programming Languages Used

- HTML
- CSS
- JavaScript
